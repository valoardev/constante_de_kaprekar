# constante_de_kaprekar
Desafio da constante de Kaprekar, feito em Python, com somente com funções e laços simples.

# 1. Contextualização
Em 1949, o matemático indiano Dattatreya Ramchandra Kaprekar descreveu uma propriedade notável dos números de quatro dígitos: ao reorganizar os dígitos de um número qualquer (que não seja um repdigit) em ordem decrescente e em ordem crescente, e em seguida subtrair o menor do maior, repete-se o processo com o resultado até que se alcance invariavelmente o valor 6174, conhecido como a Constante de Kaprekar.

Esse processo, denominado Rotina de Kaprekar, converge em no máximo sete iterações para qualquer número elegível de quatro dígitos. A atividade proposta explora esse conceito para exercitar habilidades de lógica de programação, incluindo validação de entrada, manipulação de dígitos por operações aritméticas e estruturas de repetição.

# 2. Projeto
Desenvolver um programa que, dado um número de quatro dígitos fornecido pelo usuário, execute a Rotina de Kaprekar e exiba todo o passo a passo até atingir a constante 6174, incluindo mensagens de validação quando o dado de entrada for inválido.

# 3. Restrições
- Utilizar somente funções, tipos e laços simples (While, For, If, Else, Elif, Variáveis, ...)
- Aceitar somente números inteiros, positivos, 4 digitos e aceitar somente números com no máximo duas repetições.
